# W3lcome - Challenge 

## Monorepo
This monorepo contains the following applications:
* API ``./apps/api``
    -  Restfull API in express JS with Typescript
* Web ``./apps/web`` 
    - frontend to consume api in NextJS with Typescript

Common scripts for all applications:

```
npm run dev
```

# API
